# Line-Clearing-Mine-Charge
A simple ArmA 3 Script that creates a Line-Clearing Mine Charge with changeable distance (50 to 100 meters). A warning sound when armed and helpers for fast and easy placement.

![Mine-clearing Line Charge](https://github.com/user-attachments/assets/80890d0d-3f94-415a-b8c8-65cf6dd6bf25)


To try this out for yourself, just place down an object that will work as a spawner. Than in the init of this object write '[this] execVM "scripts\spawn_box.sqf"'. This object will now have an action allowing the spawning of the MLMC Boxes. Once spawned it can be carried around or be stored inside of an vehicle, ready for transport.
